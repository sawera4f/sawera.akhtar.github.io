Projets personnels

Création d’un bras exosquelette à porter (Université Paris 8)

Contexte : Projet universitaire avec le département génie mécanique pour concevoir un prototype d’aide au portage d’objets lourds.

Description : Conception et réalisation d’un exosquelette motorisé pour le bras, intégrant des capteurs de position.

Technologies / Outils : Arduino, servomoteurs, capteurs de flexion, impression 3D.

Résultats : Prototype fonctionnel capable de soulager jusqu’à 5 kg en statique, présenté lors de la soutenance semestrielle.

Application Android de contrôle via Wi-Fi

Contexte : Développement d’une application pour piloter à distance le prototype d’exosquelette via Wi-Fi.

Description : Création de l’interface graphique, communication asynchrone avec Arduino, gestion des commandes de mouvements.

Technologies / Outils : Android Studio (Java), Volley, protocoles TCP/IP, Arduino UNO.

Résultats : Contrôle en temps réel du prototype depuis un smartphone, validé lors d’une démonstration en laboratoire.

Intégration capteurs, servos et Arduino

Contexte : Projet visant à améliorer la précision du mouvement de l’exosquelette.

Description : Implémentation d’un système de capteurs (accéléromètre, gyroscope) pour mesurer l’orientation du bras et ajuster le mouvement des servomoteurs.

Technologies / Outils : Arduino Mega, capteurs MPU-6050, servomoteurs, code en C/C++ embarqué.

Résultats : Précision de mouvement augmentée de 30 %, meilleur confort de portage.
